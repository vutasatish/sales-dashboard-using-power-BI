# Sales Dashboard Project

## Project Description
This project provides a comprehensive sales dashboard built using Power BI, allowing users to visualize and analyze sales data effectively. The dashboard is designed to help businesses make informed decisions based on real-time data insights.

## Dataset Fields
- **Order ID**: Unique identifier for each order.
- **Customer ID**: Unique identifier for each customer.
- **Product ID**: Unique identifier for each product.
- **Quantity Sold**: Number of units sold.
- **Order Date**: Date when the order was placed.
- **Sales Amount**: Total amount generated from the sales.
- **Region**: Geographic area of the sale.

## Dashboard Tasks
1. **Visualize Sales Trends**: Track sales performance over time through various visualizations like line charts and bar graphs.
2. **Customer Analysis**: Identify key customers based on sales volume and frequency.
3. **Product Performance**: Analyze which products are performing well and which need attention.
4. **Sales by Region**: Break down sales performance by different geographic regions.

## Features
- Interactive Filters: Users can select specific time periods, products, or regions to tailor the dashboard to their needs.
- Real-time Data Refresh: Ensures that users always have the latest data available for analysis.
- User-friendly Interface: Easy to navigate for users with varying levels of data analysis experience.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vutasatish/sales-dashboard-using-power-BI.git
   ```
2. Open the Power BI file in Power BI Desktop.
3. Connect to the dataset by following the prompt to specify file paths.
4. Adjust the dashboard visuals as needed based on business requirements.
5. Publish the dashboard to Power BI Service if required.
